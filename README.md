## SeeR

Angular v1.4.0 SPA.

====

#### Description:

SPA for reading comics. SeeR uses Manga Scraper API that gives reach to two hosts (mangareader.net and mangafox.me).
Offline mode is not supported at the moment.

====

#### Setup:

```
> Development:
$ npm install
$ gulp serve


> Production:
$ npm install
$ gulp build
```
